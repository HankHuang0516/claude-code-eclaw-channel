## Lessons Learned
1. **Debug logging 加滿** — Kanban 503 花 2 小時排查，加好 log 就能秒定位
2. **i18n 每個 PR 必檢** — 2150 個缺失是累積的流程疏失
3. **前後端 key 名稱要一致** — assigned_bots vs assignedBots 導致 Unassigned bug
4. **SQL parser 要小心** — startsWith('--') 過濾掉了 CREATE TABLE
5. **Railway CI 自動部署** — git push 就會觸發，不用手動
6. **Web/App 要對齊** — 老闆要求功能同步
